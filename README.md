# Athos

> Um sistema operacional inteligente para negócios dentro do Claude Code.

Athos organiza memória, estratégia, identidade, processos e habilidades especializadas em um único ambiente de trabalho. A estrutura foi reformulada para ser profissional, modular e adaptável a diferentes contextos de negócio.

---

## O que o Athos oferece

- contexto operacional centralizado em `_memoria/`
- diretrizes visuais em `identidade/`
- fluxos especializados via skills em `.claude/skills/`
- organização para projetos, clientes e entregas
- estrutura preparada para documentação e publicação profissional

---

## Estrutura principal

- `.claude/skills/` — skills operacionais e automações
- `_memoria/` — memória empresarial e estratégia
- `identidade/` — padrão visual e diretrizes
- `docs/` — documentação técnica e operacional
- `scripts/` — utilidades e rotinas
- `templates/` — modelos reutilizáveis

---

## Instalação

1. Clone o repositório em uma pasta local.
2. Abra o projeto no VS Code.
3. Inicie o Claude Code na pasta do projeto.
4. Execute `/instalar` para montar o contexto inicial.

Exemplo:

```text
Clona o https://github.com/theylorfreitas/Athos na pasta atual,
entra nela e roda o /instalar.
```

---

## Primeiro uso

Após a instalação, os comandos principais são:

- `/abrir` — carregar contexto da sessão
- `/mapear-rotinas` — identificar tarefas repetitivas
- `/novo-projeto` — estruturar um novo projeto ou cliente
- `/salvar` — salvar alterações localmente e publicar somente quando o remote estiver configurado corretamente
- `/atualizar` — revisar o contexto e atualizar a memória

---

## Segurança

Não salve segredos, tokens, chaves ou dados privados no repositório. O projeto inclui proteção básica via `.gitignore` para arquivos como `.env`, chaves e pastas sensíveis.
